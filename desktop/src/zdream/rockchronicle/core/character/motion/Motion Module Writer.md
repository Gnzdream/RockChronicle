
#	行动模块的设置参数说明

说明在 MotionModule 领域中，所有开放的 setJson 调用。

### first = box

*	velocity
	<br/>object {x:double | undefined, y:double | undefined}
	<br/>设置速度
	<br/>支持的派生类: SingleBoxMotionModule

*	velocity.x
	<br/>double
	<br/>设置横坐标方向的速度
	<br/>支持的派生类: SingleBoxMotionModule

*	velocity.y
	<br/>double
	<br/>设置纵坐标方向的速度
	<br/>支持的派生类: SingleBoxMotionModule
